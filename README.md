# Blog 🌐

## Description 🖼️

This repository holds a blog with bits of useful information.

## Setup 🔧

1. Install Python and Poetry.
2. Install dependencies: `poetry install`.

## Usage 🧰

1. Enter the environment: `poetry shell`.
2. Serve the website: `make serve`.
